# QSAR-моделирование биологической активности и токсичности молекул

Данный проект посвящен разработке моделей машинного обучения для прогнозирования биологической активности и токсичности химических соединений. В ходе работы были решены задачи регрессионного анализа и бинарной классификации для трех ключевых медицинских показателей: **IC50**, **CC50** и **SI (Индекс селективности)**.
